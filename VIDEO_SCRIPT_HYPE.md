# Grid Wars — What Our Students Are Building

**Target length:** 45–75 seconds
**Tone:** Hype reel. Show what's at stake.

---

## [OPENING — Game running full speed, territories exploding across the grid, retro arcade music]

**VO:**
"Right now, our students are putting everything they've learned this semester to the test — using their object-oriented skills to build bots for Grid Wars. Dozens of bots, one grid, thousands of cells. Every step, each bot picks a direction and moves. Land on an empty cell — it's yours. Most territory at the end wins."

## [CUT TO — Code view: student class extending Bot, getNextMove highlighted]

**VO:**
"Each student writes a class that extends Bot — that's inheritance in action. They get movement, rendering, and scoring for free. They override one method: getNextMove. The engine calls it on every bot without knowing which class is running — that's polymorphism, and it's what makes the whole tournament possible."

## [CUT TO — Code view: grid parameter, position check, return direction]

**VO:**
"Each step, the engine passes the grid — a 2D array of who owns what — plus position. They look around, make a decision, return a direction. Their class can have instance variables to track state between moves — where they've been, where they're headed."

## [CUT TO — Game footage: simple bots vs. sophisticated bots, territory difference obvious]

**VO:**
"A simple bot that moves randomly? Easy. But the smart ones check their surroundings, hunt for open territory, and avoid the crowd. Some students will go way beyond that. How far they push it is up to them."

## [CUT TO — Tournament bracket, heat playing, results screen, students reacting]

**VO:**
"On tournament day, every bot hits the grid in a live showdown. Heats. Elimination rounds. One champion. This is where their coding skills really pay off — and where all those superpowers they've been building all semester come in handy."

## [CLOSING — Champion screen with confetti, then game board image]

**VO:**
"Who's going to take the grid? We find out soon."
